# ZAMKE (잠깨)

A mission-based alarm app designed to ensure users actually wake up.

## 🔥 Key Feature: Jolt Mode
A high-intensity wake-up system combining:
- Horror-inspired sound design
- Mission-based interaction
- Forced engagement to prevent snoozing

## 🎧 Sound Design
- Ajaeng (Korean traditional instrument)
- Nature sounds (forest, wind, animals)
- Warning alarms and chaos layers

## 🧠 Concept
ZAMKE forces users to fully wake up through interaction, not passive alarms.

## 📱 Platform
- iOS (Swift, SwiftUI)
